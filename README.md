## Project Overview
This project offers an opportunity to analyse transactional data to uncover trends in sales performance, item demands, and customer purchasing behaviour. The goal is to transform raw data sales data into actionable insights that support data-driven business decisions.
### Project Objective
To answer key business questions like;
- Which item generate the highest revenue?
- What are the monthly sales trends?
- Which categories perform best?
- How can the restaurant increase average order value?
- Are there seasonal sales patterns?

### Data Cleaning and Transformation Process
- Checked for duplicates
- Handled missing values
- Standardized date formats
- Converted numerical columns(sales, quantity, total sales) to appropriate data types
- Created calculated columns where necessary like customer type.
- Extracted month, day, and year from the order date
- Created pivot tables to summarise category, month, items
- Grouped data into performance tiers
- Calculated KPIs; 
     - Total revenue
     - Average order
     - Total orders
### Exploratory Data Analysis (EDA)
This analysis focused on:
- Monthly revenue trends
- Category-level performances
- Top revenue generating items
- Sales distribution across time
- Product contribution to total revenue
### Dashboard
An interactive Excel dashboard featuring:
- KPI summary cards
- Monthly revenue trend chart
- Top items bar chart
- Category performance breakdown
- Slicers for dynamic filtering

### Insights
- Sales showed seasonal patterns with peak months
- Some high-volume products had low revenue contribution, indicating pricing optimization opportunities
- Category performance varied significantly, affecting overall profitability.
- A small percentage of items generated a large portion of total revenue( pareto effect observed)
### Recommendations
- Focus marketing efforts on top performing items
- Introduce bundle or upsell strategies to increase average value order
- Adjust pricing strategy for low-margin high-volume items
- Use seasonal trends for inventory planning
